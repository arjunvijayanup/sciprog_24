# Practical 5

This folder contains the practical 05 codes

## Compile

For arctanh.c:
```bash
gcc arctanh.c -o arctanh -lm
```
For Fibonacci.c:
```bash
gcc Fibonacci.c -o fib
```
For scanf.c:
```bash
gcc scanf.c -o scanf
```

## Execute

For arctanh.c:
```bash
./arctanh
```
For Fibonacci.c:
```bash
./fib
```
For scanf.c:
```bash
./scanf
```
